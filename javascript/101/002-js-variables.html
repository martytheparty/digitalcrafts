Browser (platform) window commands can be execute via JavaScript.
  • alert() - creates an alert box.
  • blur() - removes the focus on a page.
  • close() - closes a window.
  • confirm() - prompts the user ok/cancel
  • ...
Browser (platform) DOM (document) commands can be executed via JavaScript.
  • document.createElement()
  • document.createTextNode()
  • document.getElementById()
  • ...
Browser and Node(platforms) console (document) commands can be executed via JavaScript.
